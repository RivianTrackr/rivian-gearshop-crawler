How It Works
  • Section strip — When a line in the article body contains the pattern text, that line and all lines below it are removed before the crawler computes the content hash.
  • The raw article text is still stored in full — filters only affect change detection and diff display.
  • Filter changes take effect on the next crawl run.
  • Patterns are matched case-insensitively.
