How It Works
  • Section strip — When a line in the body contains the pattern text, that line and all lines below it are removed before the crawler computes the content hash. The raw body is still stored in full; the filter only affects change detection and diff display.
  • Title exclude (offers crawler only) — When an offer's heading contains the pattern text, the entire offer is dropped at discovery time and never enters the database. Use this for page chrome like The latest offers from Rivian or Explore our current offers that show up as headings but aren't actual offers.
  • Filter changes take effect on the next crawl run.
  • Patterns are matched case-insensitively as substrings.
